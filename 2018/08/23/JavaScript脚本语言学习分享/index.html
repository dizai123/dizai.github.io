5、JavaScript代码的安全性

textContent

​ 不返回html文本，性能更好，安全性更高。返回纯文本，不带任何标签。

encodeURIComponent()

​ 对字符串替换所有非字母数字字符，安全性提高

6、PHP代码安全性

htmlspecialchars()

​ 把“ （双引号）、‘ （单引号）、< （小于）、> （大于）等转化为实体

trim(string,charlist)

​ 去掉字符串两端预定义的字符，默认去掉

​ “\0” (NULL)

​ “\t” (制表符)

​ “\n” (换行)

​ “\x0B” (垂直制表符)

​ “\r” (回车)

​ “ “ (空格)

strip_tags()

​ 从字符串中去除 HTML 和 PHP 标记

addslashes(string)

​ 向字符串中的预定义字符添加反斜杠，默认

​ 单引号（’）

​ 双引号（”）

​ 反斜杠（\）

​ NULL
